# git command
                                 
------
                                        
### git config 配置                           
```                                           
git config --global user.name "YourName"      
git config --global user.email "YourEmail@xxx.com"                                          
```                                           
#### git add
`git add + 文件名/目录名 命令`

#### git commit
` git commit -m "first commit"`
